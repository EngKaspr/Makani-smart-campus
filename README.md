
# Makani - PSAU Smart Campus Platform 🚀

**Makani** is a comprehensive "Super App" designed specifically for Prince Sattam bin Abdulaziz University (PSAU) students. It aims to transform the university experience into a smart, sustainable, and highly interactive journey by combining academic, navigational, and community services into a single unified platform.

This project was developed as a technical prototype for university hackathons and competitions.

## ✨ Key Features

- 🗺️ **Smart Navigation:** An interactive map powered by a smart algorithm to decode complex room numbers (e.g., C206 01 54 C), featuring real-time live indicators for parking congestion and campus events.
- ♻️ **Erth Market (Legacy Market):** A sustainable marketplace to exchange, donate, or sell used university textbooks and engineering/medical tools, promoting sustainability and reducing financial and environmental waste.
- 📢 **Reports & Lost Items:** A community-driven system to document lost and found items, as well as a quick reporting tool for campus facility maintenance.
- 🤖 **Makani AI Assistant:** An AI-powered virtual coach and academic guide that answers student inquiries regarding complex university regulations and systems through a simplified "Knowledge Magazine."
- 🤝 **Student Footprint:** A dedicated portal that utilizes students' technical, organizational, and media skills in projects serving the university and the broader community (e.g., organizing graduation ceremonies, volunteering for Hajj).

## 🛠️ Tech Stack

- **Frontend:** HTML5, Vanilla JavaScript, Tailwind CSS
- **Icons:** Lucide Icons
- **Design:** Responsive Web Design (Single Page Application Layout)
- **AI Ready:** The system is pre-configured for seamless API integration with Large Language Models (LLMs).

## 🚀 How to Run

This project is built with core web technologies and requires no complex installation:
1. Download the project files or clone the repository.
2. Open the `index.html` file in any web browser.
3. Enjoy the Makani platform experience!
